# Healthcare-operations
📌 Project Overview

The Healthcare Insights Dashboard is an interactive data visualization project built using Microsoft Power BI. The dashboard provides insights into patient demographics, hospital admissions, billing patterns, medical conditions, and insurance provider trends.

This project aims to demonstrate how healthcare data analytics can support decision-making in hospitals by identifying patterns in patient admissions, treatment results, and financial metrics.

🎯 Objectives

Analyze patient admission types (Elective, Urgent, Emergency).

Identify average hospital stay by medical condition.

Understand billing amount trends and healthcare costs.

Examine health conditions based on test results.

Compare billing distribution across insurance providers.

💡 Key Insights

Patient admissions are fairly evenly distributed among elective, urgent, and emergency categories.

Asthma patients show the longest average hospital stay compared to other conditions.

Billing amounts are similar across different insurance providers, indicating balanced insurance coverage distribution.

Test results are almost evenly split between normal, abnormal, and inconclusive categories.
